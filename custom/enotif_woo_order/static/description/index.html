Enotif WooCommerce Order

This module automatically creates a new order on the Odoo website just in few minutes after it was registered on the WooCommerce website.

Usage

1) Install the Notify Odoo wordpress plugin.
2) Specify URL of your Odoo website in it.
3) Activate WordPress REST API and get the keys.
4) On the Odoo website install this module.
5) Specify your WooCommerce URL and the secret keys.
6) Try to make a new order on WooCommerce and wait. It should appear on the Odoo website in few minutes.

Features

  • Receives a notification of a new order from WooCommerce and then save the order on the Odoo website.
  • Can import all orders from WooCommerce to Odoo.

Requires

  • All products from the WooCommerce website should exist on the Odoo website. And they should have the same SKU. If any product SKU is not found during creation of the new order then this module will create a new product with just title and price.
  • The enotif_woo_customer module is required to create a new customer. Without this module it will create the order with a default customer.
