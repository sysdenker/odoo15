Advanced Website Cart settings

Website Cart Settings make the management of cart easy and make the web shop more user friendly. Admin is provided with different rights to manage the cart with the help of different settings. Admin can define the minimum value of the cart which will abide the customer to make the order below the defined value. Customer can see the subtotal of the order lines. This module helps to set the page redirection either to same page or to cart summary.

Salient Features

  • This module provides additional cart settings to make ecommerce more attractive.
  • Subtotal menu to display the total of each orderline in the cart before and after discount.
  • Option where to redirect the page after adding product to the cart.
  • What should be the minimum cart value to validate order
  • Configuration Settings for setting the values of these options.
