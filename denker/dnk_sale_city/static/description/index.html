Denker - Ciudad de Venta

Grupo Denker, www.grupodenker.com

  •    Módulo para agregar la ciudad donde se origina la venta.

Se crea el módulo:
[dnk.sale.city] - Sale City :
Y agrega los siguientes campos:

[account.move] - Invoices :
[dnk_city_id]- Sale City

[res.user] - User :
[dnk_city_id]- Sale City

[sale.order] - Sale Order :
[dnk_city_id]- Sale City

[sale.order.line] - Sale Order Line :
[dnk_city_id]- Sale City
