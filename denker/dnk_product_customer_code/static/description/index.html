Denker - Customer Product Code

Grupo Denker, www.grupodenker.com

  •    Módulo para agregar códigos de producto por cliente.


Se redefinen los siguientes campos en Product Template:

  • [seller_ids] Vendors
  • [variant_seller_ids] Vendors

Se crea el siguiente campo en Product Variant para unir con Supplier Info:
  • [dnk_customer_ids] - Customers

Se agrega el siguiente campo en SupplierInfo para filtrar códigos de vendedores y clientes:
  • [dnk_only_code] -Only Customer Code
